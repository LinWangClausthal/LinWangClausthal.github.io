在每个函数前加装饰器就可以使用

装饰器就是为了批量实现函数额外的功能的。既可以保持函数的功能不变，又可以在调用装饰器时实现额外功能。也方便批量修改。

f = decorator(test_2) #完全等价于装饰器@的写法
def test_2():
return random.random()10
def decorator(func):
def wrapper(args, **kwargs):

    #do sth
    return round(func(*args, **kwargs),3)
#do sth
return wrapper
