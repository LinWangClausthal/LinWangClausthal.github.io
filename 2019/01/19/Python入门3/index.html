交换值

a = 15
b = 20
a,b = b,a
print (“a is {}, b is {}”.format(a,b))

命名规范

标识符的第一个字符必须是英文大小写字母或下划线

标识符的其他字符可以由英文大小写字母或下划线或阿拉伯数字组成

标识符大小写敏感

代码规范 建议

  • 尽量不要使用单字符
  • 变量名字清晰表达变量所代表的意思
  • 合理使用下划线
    notebook里面用 -+空格 来表示前面的圆点
